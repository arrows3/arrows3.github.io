Fusion 360

Tutoriales fusion 360

Hand:

A partir de un modelo 3d de un brazo, realizamos una ferula o accesorio.

  • Primero creamos 3 planos de corte para dividir la geometria en 3 partes, antebrazo, muñeca y dedos.
  • Segundo dibujamos curvas que simulen la forma de cada corte, ojala lo mas cercanas al modelo del brazo posible.
  • En tercer lugar realizamos un loft con esas 3 curvas y acomodamos las caras para que sea mas fiel al modelo.
  • Finalmente cortamos el espacio para el dedo y le damos espesor con thickness.

Sketch y patrones:

A partir de un dibujo realizaremos un módelo parametrizado de un invernadero o hangar.

  • En primer lugar vamos a insertar la imagen en fusion.
  • Dibujaremos todas las lineas del dibujo con las medidas correspondientes.
  • Una vez realizadas las lineas base, crearemos un pipe con el grosor de 30mm.
  • A estas geometrias las replicaremos con un patron rectangular.
  • Finalmente crearemos la superficie que cubre el invernadero y le añaderemos un material acrilico transparente.

Piping y Simulación:

  • Comenzamos por dibujar un plano con divisiones en el eje x,y para hacer una cuadricula.
  • Tomando una linea de pormedio seleccionamos algunas para elevarlas deformar nuestra geometria a modo de ondas.
  • Luego vamos a crear un sketch para dibujar nuevas lineas en ambos ejes, y las replicaremos usando la cuadricula rectangular.
  • Estas nuevas curvas las proyectaremos sobre nuestra superficie de ondas.
  • Al resultado de estas cruvas le haremos un pipe, al cual luego procederemos a hacer un estudio de carga.
  • para el estudio de carga debemos seleccionar los puntos de constraints (puntos que iran anclados) y los puntos de loads (puntos donde se aplicará la carga)
  • Realizamos solve study y el resultado es la deformacion de la geometria al recibir esas cargas.

Extrucción y cortes en revolución:

  • A partir de otro dibujo realizaremos un objeto que debe tener orificios en forma helicoidal.
  • Para esto haremos primero un circulo el cual extruiremos para la base.
  • Luego haremos un borde de goma con un pipe y otro plano sobre este para una tapa.
  • Crearemos el cilindro que tendra los orificios y una espiral en este mismo, la cual sera nuestro plano guía para hacer los corte.
  • Haremos cortes creando esferas y un patrón en un path(este path es la curva de la espiral)
  • finalmente Realizaremos otro patron pero esta vez circular, para repetir los corte que ya teniamos.
