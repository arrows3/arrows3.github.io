Software Creativo: Creando un personaje en Maya, Photoshop y Keyshot.

Tutorial completo:

Etapa 1: Modelado

  • primero comenzamos definiendo lo que queremos modelar, para poder determinar a partir de que geometría basica lo haremosE en nuestro caso usaremos un cubo, que a partir de subdivisiones iremos modificando hasta llegar a la forma deseada. usaremos operaciones como: Insert Edge, Extrude Surface y Smooth Geometry.

Etapa 2: UV Mapping

  • El UV Mapping es proyectar un plano bidimensional en un objeto tridimensional, de esta manera podremos aplicar texturas en un futuro. Para esto debemos mapear nuestro objeto para ver claramente las partes que lo componen y poder exportar un archivo .psd para Photoshop.

Etapa 3: texturizado

  • Una vez realizado el mapeo podemos trabajar en Photoshop para crear las texturas, colores y detalles de nuestro personaje. Al terminar este proceso volvemos a Maya para aplicar las texturas y ver si el resultado es el esperado.

Etapa 4: renderizado

  • Finalmente exportamos nuestro personaje parte por parte en archivos .obj, los cuales transfieren el color y el UVMap hacia Keyshot. Una vez importemos los archivos, podemos empezar a jugar con los diferentes materiales, luces y ambientes para obtener un resultado realista.
